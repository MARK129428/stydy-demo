CSS 选择器笔记 (清晰版)

1. 基础选择器

.highlight { color: red; }
#special { color: blue; }

这是类选择器 .highlight

这是ID选择器 #special

2. 组合选择器

div p { color: green; }
div > p { background: #d0f0f0; }
h1 + p { color: orange; }
h1 ~ p { font-style: italic; }

标题

紧跟标题的段落（h1 + p）

同级其他段落（h1 ~ p）

div > p 选中

这个不会被选中（不是直接子元素）

3. 属性选择器

input[type="text"] { border: 2px solid blue; }
input[type="password"] { border: 2px solid red; }

4. 伪类选择器

a:hover { color: red; }
li:first-child { color: red; }
li:last-child { color: blue; }
li:nth-child(odd) { background: #f8f9fa; }
li:not(.skip) { font-weight: bold; }
这是一个链接（试试 hover、click）
  • 第1个 (first-child)
  • 第2个
  • 第3个 (not 规则排除)
  • 第4个
  • 第5个 (last-child)

5. 伪元素选择器

p::before { content: "👉 "; }
p::after { content: " 👈"; }
p::first-letter { font-size: 1.5em; color: red; }
p::selection { background: yellow; }

这是一个演示伪元素的段落
