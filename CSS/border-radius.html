CSS border-radius 笔记示例

1. 基本圆角

border-radius: 10px

2. 单边圆角

top-left
top-right
bottom-left
bottom-right

3. 椭圆形圆角

border-radius: 50px / 20px

4. 多值组合

10px 30px 50px 70px

5. 与阴影配合

圆角 + 阴影

说明：

  • border-radius 可设置单边、多个值或椭圆形圆角
  • 语法：
    • 单值：所有角统一圆角
    • 四值组合：上左、上右、下右、下左
    • 椭圆：水平半径 / 垂直半径
  • 可与阴影、边框等配合增强视觉效果
