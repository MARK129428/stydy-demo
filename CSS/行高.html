CSS 行高与图片对齐笔记

1. 行高基础示例

line-height: normal;
默认行高，通常约 1.2~1.4 倍字体大小
line-height: 1.5;
常用推荐行高，阅读更舒适
line-height: 40px;
固定行高，不随字体大小变化

2. 行高与图片的关系

图片默认是 inline 元素，会受行高影响，默认与文字基线对齐。

baseline 对齐
middle 对齐
top 对齐
bottom 对齐
text-top 对齐
text-bottom 对齐
block 元素，不受行高影响

注意：

  • 当图片在行内时，如果 line-height 比图片小，会撑高整行。
  • 底部空隙来源于 inline 元素默认 baseline 对齐。
  • 解决方式：
    • 使用 vertical-align 调整图片位置（如 middle/top/bottom/text-top/text-bottom）
    • 将图片改为 display: block
    • 使用 flex 布局让图片和文字垂直居中
