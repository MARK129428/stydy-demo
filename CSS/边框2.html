CSS 边框使用图片笔记示例

1. 九宫格概念与 border-image

九宫格通过将边框分为九个区域（四角、四边、中间）来显示图片边框，CSS3 的 border-image 实现了同样效果，无需额外 div 或表格。

2. border-image 属性控制

指定边框宽度、透明边框，并用图片作为边框。

border-image repeat 示例（图案重复填充）
border-image round 示例（图案整除填充，避免边缘不完整）

说明：

  • border-width：定义边框宽度，对应图片切片大小。
  • border-style：必须指定，否则 border-image 无效。
  • border-color：可透明，显示图片边框。
  • border-image-slice：切割图片九宫格区域。
  • repeat：边框图案重复填充边框长度。
  • round：边框图案整除填充，避免边缘不完整。
