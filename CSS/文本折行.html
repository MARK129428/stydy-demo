CSS 文字折行属性笔记示例

1. white-space 属性

white-space: nowrap;
文本不会换行，即使超出容器宽度也会延伸。
white-space: pre;
保留空格和换行，类似 pre 标签显示。
white-space: normal;
默认折行方式，文本遇到容器边界换行。

2. word-break 属性

word-break: break-all;
长单词或 URL 可以随意断行。 超长文字测试：abcdefghijklmnopqrstuvwxyz1234567890
word-break: keep-all;
中文文字不会在单词中间断行，只在空格或标点处换行。 中文测试文本：这是一个很长的中文句子用于测试换行效果。

3. overflow-wrap (word-wrap) 属性

overflow-wrap: break-word;
当单词太长时，允许在必要时换行以防止溢出。 超长单词测试：averylongwordthatiswaytoolongtobefitinline
